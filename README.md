 Tailwind CSS

This project is a responsive App & Software landing page hero section built using HTML and Tailwind CSS  
It recreates a modern SaaS-style hero layout similar to professional templates, focusing on clean design, layout structure, and responsiveness.

The hero section includes:
- A headline with highlighted text
- Short descriptive paragraph
- Call-to-action buttons 
- Apple logo integration
- Phone mockup image
- Decorative UI elements (circles, sparkles, waves)
- Fully responsive layout for mobile and desktop screens
